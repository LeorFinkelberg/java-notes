У стандартного процесса Java есть и стек, и куча. В стеке хранятся локальные переменные. Если они содержать примитивы, то примитивное значение напрямую размещается в стеке. 

С другой стороны, локальные переменные _ссылочных типов_ указывают на область в _куче_, где фактически буду создаваться объекты. Память для примитивных полей объекта по-прежнему выделяется по адресам внутри кучи.
